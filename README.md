-ConversorDeMonedas

Una aplicación en Java que permite convertir cantidades entre diferentes monedas. Este proyecto fue desarrollado como parte del ONE Challenge (desafío de aprendizaje práctico) y tiene por objetivo poner en práctica conceptos de programación orientada a objetos y estructura de datos básicos.

-Descripción

Este programa solicita al usuario un monto y dos monedas (origen y destino), y luego calcula el valor convertido según las tasas de cambio definidas en el código. Se creó como una solución simple al desafío de conversión de monedas, ideal para practicar lógica, estructuras condicionales y manejo básico de entradas por consola en Java.

-¿Cómo funciona?

El usuario ingresa una cantidad a convertir.

Elige la moneda de origen (por ejemplo, MXN).

Elige la moneda de destino (por ejemplo, USD).

El programa calcula y muestra el resultado de la conversión.

Opcionalmente se puede repetir otra conversión sin cerrar la aplicación.

Nota: Las tasas de cambio se almacenan dentro del código (o en estructuras del proyecto), por lo que no dependen de una API externa en tiempo real en esta versión inicial.

-Tecnologías usadas

Este proyecto está desarrollado en:

Java — Lenguaje principal del programa.


-¿Cómo ejecutar el proyecto?

Clona el repositorio:

git clone https://github.com/Arturo-M-Pedroza/conversorDeMonedas.git

Abre el proyecto en tu IDE favorito.

Compila la aplicación.

Ejecuta la clase Main.java.

Sigue las instrucciones en consola para realizar conversiones.

-Ejemplo de uso
Ingrese la cantidad:
100
Seleccione la moneda origen:
MXN
Seleccione la moneda destino:
USD
Resultado: 5.30 USD

(Los valores de conversión pueden variar según cómo se implementen las tasas en el código)
